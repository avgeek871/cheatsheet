Tables

Tables can be tedious to set up but can be a useful way to display information. Let's look at the process:

Tags You'll Need:

  • <table>: This defines the entire table element. Your entire table should be enclosed in this tag.
  • <thead>: This defines the header of the table, if you'd like to include one. Be sure to enclose the entire header with this tag.
  • <tbody>: This defines the body of the table. This should include all of your data, except the header/footer.
  • <tr>: This defines each row of your table. This should contain at least one of the next tag for each instance.
  • <td>: This defines each column within the <tr> tags for each row.
  • <tfoot>: This defines the footer of the table, if you'd like to include one. Again, be sure to enclose the entire footer with this tag

Don't forget to close each tag!

Tags

Tags enable you to use all the different functionality of HTML! There are hundreds of different tags, so we'll only cover a few of the most common here. All tags require closing, unless specifically noted as 'self-closing'. There are other resources linked below which cover many more.

Headings:

There are six different Headings you can use in HTML. These enable you to separate and organize your information into clear sections, and can also allow you to specify specific formatting for each type. They are called as follows:

  • <h1>
  • <h2>
  • <h3>
  • <h4>
  • <h5>
  • <h6>

These are all extremely similar, the only difference being the size each appears in the default HTML format. Obviously, with CSS you could make an <h6> tag appear larger and more prominent than the <h1> heading if you wanted to! But this would be needlessly confusing, so I wouldn't recommend it.

General Text:

General text can be inserted with the <p> tag. Within that tag, any text will appear exactly as typed. There are a number of formatting options available within CSS again, but what if you want to only apply it to a small portion of the text? Let's look at some common formatting:

  • <a>: This tag will create a link around any text which is put between the tags. A href="http://www.example.com" property must be specified.
  • <strong>: This tag will create bold text.
  • <em>: This tag will create italics text.
  • <span>: This tag will create underlined text.
  • <!--text-->: This will create a comment within HTML. Anything within the tag will be ignored by the browser. Be sure to close this tag in the correct place!
  • <hr>: This will create a single horizontal line, usually as a formatting choice to break up sections (self-closing).
  • <br>: This will create a single blank line, usually as a formatting choice to break up sections (self-closing).

Lists:

HTML allows us to create two types of lists: ordered (<ol>) and unordered (<ul>). The only difference is that one uses un-numbered bullets, the other specifies a number/letter/etc for each. Each item on the list must be enclosed by the <li> tag.

Multimedia:

Every good website has some sort of multimedia! Pictures and videos can really make or break a website. Fortunately, they're pretty easy to add in.

Images

Images can be inserted into your webpage using the <img> tag (self-closing). There are several attributes which can be specified, but the only required one is src="http://www.example.com", which gives the source of the image. It is good practice to include an alt="example text" attribute as well, which tells the browser what text to display in case the image can't be displayed.

Audio

Audio can be added to your webpage using the <audio>; tag. Within the opening tag, you can add controls if you'd like the user to be able to control the audio. Inside the tag, there should be a <source> tag which, as you may guess, specifies the source of the audio. Here's an example of this tag's use:

<audio controls>
<source src="example.mp3" type="audio/mpeg">
No audio support.
</audio>

In this example, the first line contains the opening tag and the controls attribute, allowing the user to control the audio. The second line specifies the source and its' type. The third line tells the browser what to display if the audio is not playable for any reason. And the last line is the closing tag.

Video

Video is very similar to audio; it uses the <video> tag. Again, within the opening tag a controls attribute can be added. New with video: width and height attributes can be specified in pixels. Let's look at an example:

<video width="500" height="300" controls>
<source src="video.mp4" type="video/mp4">
No video support.
</video>

In this example, the first line contains the video tag, attributes for the width and height of the video, and specifies controls should be provided. The second line again provides the source and type of file. Third, what the browser should display if the video cannot be loaded for some reason. And fourth, the closing tag.
